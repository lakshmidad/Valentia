> **"Connecting hearts, one secure link at a time."**

A web platform that transforms ordinary messages into interactive digital surprises by generating secure, personalized Valentine message links with animated countdown reveals and multimedia memories.

## üåü Project Overview
HeartLink is a personalized Valentine-themed web application that allows users to create private, shareable message links for someone special. Instead of ordinary text, users craft a beautifully designed digital surprise with heartfelt text, themes, music, and a timed reveal.

### ‚ù§Ô∏è Core Features
- **Personalized Message Creator**: Simple interface to input sender/recipient names and the core message.
- **Unique Secret Links**: Auto-generated, secure, shareable URLs (e.g., `heartlink.com/m/xyz123`).

### üíé Advanced Features
- **Multimedia Support**: Upload a photo or a short voice note to accompany the text.
- **Celebratory Effects**: Confetti bursts, floating hearts, and particle effects upon reveal.
- **Theme Selector**:
    - *Cute*: Pastel colors, doodles, playful fonts.
    - *Romantic*: Deep reds/pinks, elegant serif fonts, glow effects.
    - *Minimal*: Clean lines, monochrome/gold accents, modern typography.
- **Digital Keepsake**: "Download as Image" button to save the message card.
- **Engagement Metrics**: Simple view counter (sender can see if it's been opened).
- **Privacy & Security**: Auto-expiring links (e.g., "Delete after 24 hours" or "Delete after 1 viewing").
